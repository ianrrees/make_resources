make

make is a program that can help you organise building programs. It is typically used on UNIX-ish systems, for building programs in compiled languages like C.

It has been an industry standard tool for decades.

Tips

Plenty of guides online - a couple things I find handy:

  • Use -d and/or -n to see what's going on
  • Use -jN to build in parallel, leave it off for debugging
  • Prepend @ to suppress console output
  • Can have multiple targets:

cryptic-filename.hex firmware: main.c
<tab>@echo "Building firmware"

Deugging

  • -j1 (!)
  • -n
  • verbose=1

From the manual: You can write recursive make commands just by copying this example, but there are many things to know about how they work and why
